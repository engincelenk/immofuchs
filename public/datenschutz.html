1. Verantwortliche Stelle

Privatperson (kein Unternehmen, keine kommerzielle Tätigkeit).
Kontakt: info@immofuchs.info

2. Datenverarbeitung auf einen Blick

Immofuchs ist eine rein clientseitige Webanwendung. Alle Berechnungen finden ausschließlich in Ihrem Browser statt. Es werden keine personenbezogenen Daten an Server übertragen.

3. Lokale Datenspeicherung (localStorage)

Ihre Eingaben (Kaufpreis, Zinssatz, etc.) werden im localStorage Ihres Browsers gespeichert, damit Sie beim nächsten Besuch fortfahren können. Diese Daten:

  • verlassen niemals Ihren Browser
  • sind nur für Sie zugänglich
  • können jederzeit über die Browser-Einstellungen gelöscht werden

4. Cookies

Immofuchs setzt keine Tracking-Cookies. Es wird lediglich localStorage verwendet (technisch notwendig). Der Cookie-Consent-Eintrag wird in localStorage gespeichert, sobald Sie eine Auswahl getroffen haben.

5. Hosting & Server-Logs

Die Website wird bei einem Hosting-Anbieter betrieben. Beim Abrufen der Seiten werden durch den Hosting-Anbieter automatisch technische Zugriffsdaten in Server-Log-Dateien gespeichert (Browsertyp, Betriebssystem, Referrer-URL, Datum/Uhrzeit, IP-Adresse). Diese Daten werden aus technischen Gründen erhoben, nicht mit anderen Quellen zusammengeführt und dienen allein dem sicheren Betrieb. Rechtsgrundlage: Art. 6 Abs. 1 lit. f DSGVO.

6. Externe Dienste

Zur Darstellung der Website werden Schriftarten von Google Fonts (DM Sans) geladen. Dabei kann die IP-Adresse an Google übertragen werden. Details: Google Datenschutzerklärung.

Optional werden tagesaktuelle Bauzinsen von einer öffentlichen JSON-Datei geladen (GitHub Pages). Diese Datei enthält keine personenbezogenen Daten; beim Abruf wird jedoch Ihre IP-Adresse an GitHub übertragen. Details: GitHub Privacy Statement.

Es werden keine weiteren externen Dienste eingebunden — kein Google Analytics, keine Werbung, kein Facebook Pixel, keine Affiliate-Links.

7. Kein Kontaktformular

Diese Website enthält kein Kontaktformular. Eine Kontaktaufnahme ist ausschließlich per E-Mail möglich. Wenn Sie uns per E-Mail kontaktieren, speichern wir Ihre Angaben zur Bearbeitung der Anfrage. Eine Weitergabe an Dritte erfolgt nicht. Rechtsgrundlage: Art. 6 Abs. 1 lit. f DSGVO.
